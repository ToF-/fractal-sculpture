Documentation

(<<-) :: AttrTag -> Text -> Attribute infix 4 #

Infix version of bindAttr Each argument is set using <<-, the bindAttr function or ->>.

[Width_ <<- "100%", Height_ <<- "100%", Fill_ <<- "red"]

(->>) :: Text -> AttrTag -> Attribute infix 4 #

Infix version of bindAttr with it's arguments reversed.

["100%" ->> Width_, "100%" ->> Height_, "red" ->> Fill_]

bindAttr :: AttrTag -> Text -> Attribute #

Make an Attribute from it's value constructor and it's text value. by combining an AttrTag with it's value.

[bindAttr Width "100%, bindAttr Height "100%", bindAttr Fill "red"]
